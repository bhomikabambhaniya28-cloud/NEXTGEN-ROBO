## 🚀 Features
- **ROS Integration**: Pre-configured packages for navigation, SLAM, and sensor fusion
- **AI Modules**: Object detection (YOLO), motion planning (MoveIt), and reinforcement learning
- **Hardware Support**: Arduino, Raspberry Pi, Jetson, and STM32 firmware templates
- **Simulation**: Gazebo and WebOTS-ready URDF models
- **Developer Tools**: Jupyter notebooks for prototyping + CI/CD pipelines

## 📦 Quick Start
### Prerequisites
- ROS 2 (Humble/Hydrogen) or ROS Noetic
- Python 3.8+
- OpenCV 4.0+
